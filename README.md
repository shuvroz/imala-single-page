This project is a responsive **one-page** website created with **HTML and CSS only** (no JavaScript).

## Project Overview

**IMALA** is a fictional company focused on construction planning and architectural presentation.  
The website includes:

- A hero banner (Plan)
- Three split-layout sections (services)
- A footer with links and contact info

The goal of this project was to recreate a modern professional layout using only HTML and CSS, with clean structure and responsive design.

---

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (layout + responsive)
- Git & GitHub (version control)
- GitHub Pages (deployment)

---

## Features

- One-page navigation (anchors)
- Fixed header navigation
- Hero image with overlay text
- 3 split sections (text + image layout)
- Responsive design (mobile/tablet/desktop)
- Custom logo included
- Footer navigation
